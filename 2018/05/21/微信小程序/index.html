    在mpvue中使用Vant Weapp组件库

    • 1、将下载的小程序组件的dist目录拷贝到工程static目录下，然后改名，比如van-weapp

    • 2、打开ES6转ES5

    • 3、在需要使用van-weapp组件的页面创建main.json文件，并进行引入
      然后在main.json中添加如下内容

      {
  "usingComponents": {
      "van-tab": "../../../static/vant-weapp/tab/index",
      "van-tabs": "../../../static/vant-weapp/tabs/index"
  }
}
    • 4、使用自定义组件
        <!-- 使用第三方组件 -->
  <van-tabs v-bind:active="active" @click="onChange">
    <van-tab title="标签 1">内容 1</van-tab>
    <van-tab title="标签 2">内容 2</van-tab>
    <van-tab title="标签 3">内容 3</van-tab>
    <van-tab title="标签 4">内容 4</van-tab>
  </van-tabs>
    • 注意：在vant-weapp的官方文档中的具体用法是使用wxml的语法，所以我们不能直接照搬使用
      比如官方文档示例代码为：
      <van-tabs active="{{ active }}" bind:change="onChange">
<van-tab title="标签 1">内容 1</van-tab>
<van-tab title="标签 2">内容 2</van-tab>
<van-tab title="标签 3">内容 3</van-tab>
<van-tab title="标签 4">内容 4</van-tab>
</van-tabs>
